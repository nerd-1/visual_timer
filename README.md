# visual_timer #

About:
--
Visual timer is a simple terminal based timer; inspired by the principal of user-centric design.

Usage:
--
* visual_timer [TIME]

Starts a countdown from the specified time. Must be in the format: HH:MM:SS

Installation:
--
pip install visual_timer
